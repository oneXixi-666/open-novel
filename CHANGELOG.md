# 更新记录

## 0.1.4

### 新增

- 本地 SQLite 工作区首次启动自动初始化，并提供常用写作脚手架与简单示例。
- 在线版本检测和一键更新，默认每分钟检查一次新版本。
- 发布包隐私门禁，拒绝数据库、环境文件、本机路径、访问令牌和开发产物。

### 修复

- 章节进度计算统一使用作品计划中的每章目标字数。
- 发布构建会拒绝未被 Git 跟踪的 Python 运行模块，避免更新包缺少源码。
- 预制模型模板存入 SQLite，初始化只补充缺失模板，不覆盖用户修改。

### 数据安全

- AI 账号、密钥、小说内容、数据库和本地配置不会进入 Git 或发布包。
- 开发计划、审计记录、内部验收证据和一次性测试产物仅保存在本机。
